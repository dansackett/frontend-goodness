This setup is how I like my frontend stack to be setup. It allows for quick iteration and faster development. It includes:

  • AngularJS

    AngularJS is my favorite frontend framework and allows for fast data-driven websites.

  • CoffeeScript

    CoffeeScript makes writing Javascript faster and feels like Python to me (which I love).

  • SASS

    SASS is an awesome CSS preprocessor allowing for better strictured CSS.

  • Twitter Bootstrap

    Twitter bootstrap allows you to scaffold apps fast using a community standard in development.

  • AngularJS Bootstrap

    Replaces the need for jQuery by building Bootstrap Javascript componnents in AngularJS.

  • AngularJS UI-Router

    The community darling for routing AngularJS applications.

  • Gulp

    Gulp is a build tool that allows you to watch for frontend changes and convert the above technologies into web format.

  • Bower

    Front-end package management.
